# Process Mining and Predictive Analysis Project

## Overview

This project implements a process mining and predictive analysis workflow using Python.  
It leverages event log data to extract process information and applies statistical and machine learning techniques for analysis and prediction.

---

## Features

- Event log processing using **PM4Py**
- Data manipulation with **Pandas** and **NumPy**
- Statistical analysis with **SciPy**
- Predictive modeling using **Scikit-learn**
- Error evaluation using Root Mean Squared Error (RMSE)
